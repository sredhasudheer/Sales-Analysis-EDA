**Overview**
This project performs Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python. The objective is to analyze sales performance, identify business trends, and gain insights from the data.

## Dataset
- Dataset: Sample Superstore
- File: SampleSuperstore.csv

## Tools & Libraries
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Analysis Performed
- Loaded and explored the dataset
- Checked data types and missing values
- Generated summary statistics
- Analyzed sales by category
- Analyzed sales by region
- Created charts for data visualization

## Key Insights
- Technology category generated the highest sales.
- Sales performance varied across different regions.
- Visualizations helped identify sales trends and patterns.
